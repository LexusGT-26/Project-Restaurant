# Jerry's LTB Website Color Palette

## Primary Colors

### Background Colors
- **Header Background**: `#ede0d4` (Beige/Light Tan)
- **Footer Background**: `#ede0d4` (Beige/Light Tan)
- **Menu Page Background**: `gray-50` (Light Gray)
- **Card Background**: `white`
- **Video Overlay**: `black/50` (50% opacity)

### Text Colors

#### Dark Chocolate Brown (Primary Text)
- **Dark Chocolate Brown (Active)**: `#3D2817`
  - Used for: Active navigation items, logo text, active indicator lines
- **Dark Brown (Inactive)**: `#5C4033`
  - Used for: Inactive navigation items

#### Dark Blue-Gray (Secondary Text)
- **Dark Blue-Gray**: `#28303d`
  - Used for: "Quick Links" and "Open Hours" headings in footer

#### Standard Grays
- **Gray-900**: `#111827` (Very Dark Gray)
  - Used for: Main headings, footer headings, body text
- **Gray-800**: `#1f2937` (Dark Gray)
  - Used for: Footer text, general text
- **Gray-700**: `#374151` (Medium Dark Gray)
  - Used for: Borders, social media icon backgrounds
- **Gray-600**: `#4b5563` (Medium Gray)
  - Used for: Secondary text, inactive filter buttons
- **Gray-500**: `#6b7280` (Medium Light Gray)
  - Used for: Placeholder text, less important text
- **Gray-400**: `#9ca3af` (Light Gray)
  - Used for: Icons, search icon
- **Gray-200**: `#e5e7eb` (Very Light Gray)
  - Used for: Hero section text, hover backgrounds
- **Gray-100**: `#f3f4f6` (Almost White)
  - Used for: Image backgrounds, borders

#### White
- **White**: `#ffffff`
  - Used for: Text on colored backgrounds, card backgrounds, search input background

## Accent Colors

### Tan/Beige Accents
- **Tan/Beige**: `#D3AC8B`
  - Used for: Active filter buttons in menu page, hover states
- **Light Beige**: `#ede0d4`
  - Used for: Header, footer, search button background
- **Beige Hover**: `#d9c9b8`
  - Used for: Search button hover state

### Red Accents
- **Red-600**: `#dc2626`
  - Used for: Social media icon hover states

## Usage Summary

### Header
- Background: `#ede0d4`
- Logo Text: `#3D2817` (Dark Chocolate Brown)
- Navigation Active: `#3D2817`
- Navigation Inactive: `#5C4033`
- Active Indicator: `#3D2817`

### Footer
- Background: `#ede0d4`
- Main Heading: `gray-900`
- Section Headings: `#28303d` (Quick Links, Open Hours) or `gray-900` (Contact Us)
- Body Text: `gray-800`
- Border: `gray-700`
- Social Icons: `gray-700` background, `red-600` hover

### Hero Section
- Text: `white`, `gray-200`
- Search Button: `#ede0d4` background, `#d9c9b8` hover
- Overlay: `black/50`

### Menu Page
- Background: `gray-50`
- Active Filter: `#D3AC8B` background, `white` text
- Inactive Filter: `white` background, `gray-600` text, `#D3AC8B` hover
- Menu Items: `gray-900` text
- Cards: `white` background
